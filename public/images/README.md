# 📂 Estructura de Assets - Joyas JP

## 📁 Directorios creados:

### `/images/logos/`
- `logo-main.png` - Logo principal (400x400px recomendado)
- `logo-horizontal.png` - Logo para header (300x100px)
- `logo-white.png` - Logo blanco para fondos oscuros
- `logo-icon.png` - Solo icono, formato cuadrado

### `/images/products/`
- `featured/` - Productos destacados para homepage
- `categories/` - Imágenes por categoría (anillos, collares, etc.)
- `gallery/` - Galería general de productos

### `/images/hero/`
- `hero-bg.jpg` - Imagen de fondo para hero section
- `hero-mobile.jpg` - Versión móvil del hero
- Otros assets del hero principal

### `/images/ui/`
- `patterns/` - Patrones decorativos
- `textures/` - Texturas de fondo
- `icons/` - Iconos personalizados

### `/videos/`
- `hero-main.mp4` - Video principal del hero
- `product-showcase.mp4` - Videos de productos
- `brand-story.mp4` - Video institucional

## 🎯 Rutas de uso en código:

```tsx
// Logo principal
/images/logos/logo-main.png

// Video hero
/videos/hero-main.mp4

// Productos
/images/products/featured/producto-1.jpg
```

## 📏 Tamaños recomendados:
- Logos: 400x400px (PNG con transparencia)
- Productos: 800x800px cuadradas
- Videos: 1920x1080, máximo 10MB
- Hero images: 1920x1080
